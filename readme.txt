GenomeView readme

Documentation: http://genomeview.sourceforge.net/

Download: http://downloads.sourceforge.net/genomeview/

Webstart: http://genomeview.sourceforge.net/start.jnlp.php


Import in Eclipse:
- Check out the trunk of GenomeView in the SVN
- Add all libraries in lib/ to the classpath
- Add a JUnit library
- Add javaws.jar to the classpath. This library is present in your JDK lib/ directory
- If there is no version of JAnnot in the lib/ directory, check out the JAnnot project and add it to the GenomeView classpath
